# 05 | 元类 (Metaclasses)

## 面试常见问题

- 什么是元类？`type` 和元类的关系？
- `__new__` 和 `__init__` 在元类中的作用？
- 元类的实际应用场景？
- `__init_subclass__` 和元类的区别？
- 元类能实现什么装饰器做不到的？

## 关键理解

```
类                      ← 元类的实例
对象                    ← 类的实例

type                    ← 所有类的元类（包括自己）
object                  ← 所有类的基类

创建类的过程:
1. type(name, bases, dict) ← 动态创建类
2. metaclass.__new__()     ← 拦截类创建
3. metaclass.__init__()    ← 初始化类对象
```

## 元类 vs 装饰器

| 特性 | 装饰器 | 元类 |
|------|--------|------|
| 作用时机 | 定义后 | 定义时 |
| 可修改 | 函数/方法 | 类本身 |
| 继承传播 | 不自动 | 子类也会触发 |
| 复杂度 | 低 | 高 |
