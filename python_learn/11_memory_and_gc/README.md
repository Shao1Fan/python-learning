## 关键概念

```
引用计数 (Reference Counting)
  ├── 每个对象维护 ob_ref
  ├── 增: 赋值、传参、容器添加
  └── 减: del、离开作用域、覆盖引用

循环引用 (Circular Reference)
  ├── 无法靠引用计数回收
  └── 由 gc 模块处理

分代回收 (Generational GC)
  ├── 0代: 新创建，频率高
  ├── 1代: 存活一轮，频率中
  └── 2代: 长期存活，频率低
```
