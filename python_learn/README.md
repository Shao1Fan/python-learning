```
    ↓                                              ↓
类型系统 ← 并发/并行 ← 函数式技巧 ← 魔术方法/描述符
    ↓                                              ↓
dataclass/模式匹配 → 异常/日志 → 高级集合 → 元类
    ↓
内存管理 → 打包/导入 → 面试刷题
```

## 使用方式

每个目录下的 `.py` 文件都可以直接运行：

```bash
python python_learn/01_decorators/custom_decorator.py
python python_learn/04_async_programming/basic_async.py
```

部分文件需要安装依赖（如 `aiohttp`），README 中会有说明。

## 推荐学习顺序

1. 先过 **01~04**（核心高频，面试必问）
2. 再攻 **06~10**（进阶必知）
3. 最后 **05、11~15**（深入原理 & 工程实践）
4. 刷 **interviews_top100.md** 查漏补缺
