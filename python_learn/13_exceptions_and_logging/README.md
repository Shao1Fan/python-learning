## 异常执行顺序

```python
try:
    ...         # 1. 先执行
except:
    ...         # 2. 有异常时才执行
else:
    ...         # 3. 没有异常时执行
finally:
    ...         # 4. 无论是否异常都执行
```

## 日志级别

| 级别 | 数值 | 使用场景 |
|------|------|----------|
| DEBUG | 10 | 调试信息 |
| INFO | 20 | 正常信息 |
| WARNING | 30 | 警告 |
| ERROR | 40 | 错误 |
| CRITICAL | 50 | 严重错误 |
